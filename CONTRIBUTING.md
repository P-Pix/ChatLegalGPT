# Contribuer à ChatLegalGPT

Merci pour votre intérêt !

## Démarrer
1. Forkez le dépôt
2. Créez une branche : `git checkout -b feat/ma-feature`
3. Lancez les tests / linters si vous en ajoutez
4. Ouvrez une Pull Request avec :
   - le contexte
   - ce que vous avez changé
   - comment tester

## Conventions
- Code Python : format lisible, fonctions courtes, erreurs explicites.
- API : stable, versionnée, pas de breaking change sans discussion.
- Données juridiques : privilégier les sources officielles et conserver les métadonnées (id, date, url).

## Signaler un bug
Ouvrez une issue avec :
- étapes de reproduction
- résultat attendu / observé
- logs / stacktrace si possible
